# Movie Recommendation System

This project is a Machine Learning based Movie Recommendation System that suggests movies similar to the selected movie.

The system uses content-based filtering techniques to recommend movies based on similarity scores.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Similarity Calculation
5. Movie Recommendation

## Dataset
The dataset used contains movie details such as:
- Movie title
- Cast
- Crew
- Genres
- Keywords

These features are combined to calculate similarity between movies.


## How the Recommendation Works

The system uses **Cosine Similarity** to find movies that are most similar to the selected movie.

Steps:
1. Select a movie
2. Find similarity scores
3. Sort movies based on similarity
4. Recommend top 5 similar movies
